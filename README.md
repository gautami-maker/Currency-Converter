#  Currency Converter

A sleek, modern currency converter web app that fetches live exchange rates using a public API and converts between currencies in real time with proper error handling.

##  Features
- Live exchange rate fetching using async/await
- Convert between multiple currencies
- Modern glassmorphism UI
- Loading and error states for better UX
- Fully responsive design

##  Tech Stack
- HTML
- CSS (Glassmorphism UI)
- JavaScript (ES6+)
- Exchange Rate API

##  Project Structure
currency-converter/
├── index.html
├── style.css
└── script.js


## ⚙️ How It Works
1. User enters an amount and selects currencies  
2. App fetches real-time exchange rates from an API  
3. Conversion is calculated and displayed instantly  
4. Errors (invalid input, API failure) are handled gracefully  

##  What This Project Demonstrates
- Working with REST APIs
- Async JavaScript and error handling
- Clean UI/UX design thinking
- Real-world frontend problem solving
